# Отчет по заданию 11-11-25

## 1\. Команды для сборки

Проект настроен на сборку с помощью инструмента **Vite**. В `package.json` определены следующие скрипты:

  * **Построение проекта (сборка в папку `dist`):**
    ```bash
    npm run build
    ```
  * Предпросмотр собранной версии:
    ```bash
    npm run preview
    ```

-----

## 2\. Сравнение размеров бандла

Было проведено сравнение двух подходов к подключению Bootstrap 5: полное подключение всех стилей и модульное (SCSS), где импортируются только необходимые компоненты.  

### Таблица результатов

| Тип сборки | JS Bundle | CSS Bundle | Комментарий |
| :--- | :--- | :--- | :--- |
| **Полный Bootstrap** | 154.27 kB | **226.83 kB** | Подключены все стили Bootstrap целиком (`@import "bootstrap/scss/bootstrap"`). |
| **Оптимизированный** | 154.27 kB | **109.97 kB** | Выборочный импорт SCSS (только Grid, Modal, Buttons). |

### Вывод

Благодаря выборочному импорту стилей через SCSS удалось **сократить размер CSS-файла более чем в 2 раза** (с 226 kB до 109 kB). Размер JavaScript файла остался неизменным, так как основной объем занимают необходимые зависимости (`luxon` и ядро `bootstrap`), которые требуются в обоих случаях.

## 4\. Интерфейс приложения (UI)

Реализована сетка Bootstrap с пропорциями колонок **2-8-2**, страница корректно собирается и работает локально (через `npm run preview`).

  * **Левая и правая колонки:** имеют визуальные границы для демонстрации Grid-системы.
  * **Центральная колонка:** содержит красную кнопку на всю ширину.
  * **Функционал:** При клике открывается модальное окно, где библиотека `luxon` выводит текущее время.
